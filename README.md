# 股票小组件项目

一个基于 React 的实时股票信息查看小组件，支持股票搜索、实时行情展示、自选股管理和数据自动刷新。

## 技术栈

- React 18 + TypeScript
- 新浪财经/腾讯财经免费API
- LocalStorage 本地存储

## 快速开始

1. 安装依赖
2. 运行开发服务器
3. 嵌入小组件到你的应用

详细说明请参考 [docs/USAGE.md](./docs/USAGE.md)

## API文档

请参考 [docs/STOCK_API.md](./docs/STOCK_API.md)

## 功能特性

- 🔍 股票搜索：支持按股票代码或名称搜索
- 📊 实时行情：显示当前价格、涨跌幅、成交量等
- ⭐ 自选股管理：支持添加、删除、排序
- 🔄 数据自动刷新：每30秒自动更新

## 市场支持

- 上证 (sh)
- 深证 (sz)
- 港股 (hk)
- 美股 (us)
